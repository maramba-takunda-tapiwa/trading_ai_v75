## What's Happening Behind the Scenes

1. **You add credentials via Settings tab**
   - Sent to Flask `/api/configure` endpoint
   - Saved to `/app/data/results/saxo_config.json`

2. **You click Deploy**
   - Flask `/api/deploy` endpoint called
   - Launches `live_trader_saxo_v2.py` in background thread
   - Trader reads credentials from config file
   - Connects to Saxo API (demo account)

3. **Trader starts executing**
   - Fetches EUR/USD hourly candles
   - Applies V2 breakout strategy
   - Calculates entries, exits, stops
   - Executes trades automatically
   - Writes each trade to CSV

4. **Dashboard shows data**
   - Polls `/api/trades` every 5 seconds
   - Reads live_trades_log_v2.csv
   - Calculates metrics (win rate, drawdown, PF)
   - Updates charts and tables
   - Triggers alerts on limits

5. **System monitors automatically**
   - Max drawdown < $3,000 ‚úì
   - Daily loss < $600 ‚úì
   - Win rate target > 12% ‚úì
   - Profit factor target > 1.8 ‚úì
   - Soft equity stop at 85% ‚úì
