## If Something Doesn't Appear

### Dashboard doesn't load
```powershell
# Restart everything
docker-compose down
docker-compose up -d
# Then: http://localhost
```

### Credentials won't save
```powershell
# Check logs
docker-compose logs trading-ai
# Look for error messages
```

### Trades not appearing after 1 hour
1. Check Settings tab - are credentials still there?
2. Check Alerts - any errors?
3. Check logs: `docker-compose logs trading-ai`
4. Try clicking "Refresh" button

### Deploy button not working
1. Check browser console (F12)
2. Check API: `Invoke-WebRequest http://localhost/api/health`
3. Check logs: `docker-compose logs trading-ai`
