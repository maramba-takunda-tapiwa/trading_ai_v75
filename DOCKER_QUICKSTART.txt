================================================================================
TROUBLESHOOTING
================================================================================

Dashboard shows blank/white screen:
- Wait 10 seconds
- Refresh browser (F5)
- Check: docker ps
- If not running: docker-compose up -d

No trades appearing:
- Normal if just deployed
- Wait 1+ hour for first trades
- EUR/USD hourly, no trades if no breakouts
- Check: http://localhost/api/trades for raw data

Deployment fails:
- Check logs: docker-compose logs trading-ai
- Restart: docker-compose restart
- Rebuild: docker-compose build --no-cache

Port already in use:
- Someone using port 80
- Change docker-compose.yml line: "80:80" to "8080:80"
- Then: http://localhost:8080
